# Preciion module

A [Factorio 2.1](https://www.factorio.com/) mod that adds a early way to use quality on the [Lignumis](https://mods.factorio.com/mod/lignumis) 

## New Content

Add "Precision Module", brand new Lignumis way to get gamble quality. 

## Balance

The mod is poorly suited for a classic playthrough due to its low quality probability (0.8%). The main balance is designed for rather absurd science costs of x50-x1000.

## (Maybe) later expand of mod

In the future, there is a possibility that the mod will be transformed into a combat rebalancing modification for this planet. If this happens, I will notify here.
